## Analysis

Single-file bug fix. Phase 5 added Tooltip usage across three components (AppShell nav items, OverlayControls overlay buttons, HintTooltip). Radix Tooltip requires a `TooltipProvider` ancestor in the React tree or it throws a runtime error.

`App.tsx` already had `<RadixTooltip.Provider>` but `App` was never referenced in `main.tsx` (only `RouterProvider` was used, wrapping the router directly). The fix adds `<RadixTooltip.Provider delayDuration={400}>` around `RouterProvider` in `main.tsx`.
