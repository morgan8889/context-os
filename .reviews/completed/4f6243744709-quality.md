### Issues

**[IMPORTANT] QC-P1-001 — Dual ESLint config creates environment-dependent strictness**
Files: `web/eslint.config.js` (Vite boilerplate), `web/.eslintrc.json` (T003-correct)

In ESLint 8, `.eslintrc.json` and `eslint.config.js` coexist with different activation rules.
`.eslintrc.json` uses `plugin:@typescript-eslint/strict`; the Vite boilerplate `eslint.config.js`
uses `tseslint.configs.recommended` (weaker). Result: strictness level varies by environment.

Fix: Delete `web/eslint.config.js`. The `.eslintrc.json` correctly implements T003.

**[NOTE] QC-P1-002 — `.gitignore` excludes `tests/visual/snapshots/` — blocks T072**
File: `web/.gitignore`

before T072 (Phase 7) executes the deliberate 27-snapshot baseline commit.

Fix at T072 time: Remove `tests/visual/snapshots/` from `.gitignore`, run
`playwright test --update-snapshots`, commit the 27 PNGs.
